## Decisions made

| Topic | Decision | Rationale |
|-------|----------|-----------|
| Package manager | npm | pnpm/bun not installed; npm works with Node 20 |
| Routing | Route group `(app)` for authed pages | Cleaner than per-route layouts |
| Auth | Supabase magic link (passwordless) | Matches "no passwords" privacy rule |
| Styling | Tailwind v4 + CSS custom props as Shadow tokens | Tokens in `tokens.json` mapped via `@theme inline` |
| Models | Haiku 4.5 (classify) / Sonnet 4.6 (reports) | Per cost-aware pipeline skill |
| Default theme | Dark | Matches brief; bg #0B0B10 |
